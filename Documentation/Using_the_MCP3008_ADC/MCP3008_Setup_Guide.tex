\section{What is an ADC and why do I need it?}

An Analog to Digital Converter (ADC) is a device that can measure an analog voltage and covert the measurement to a digital code that can be used by a microcontroller such as the Raspberry Pi. We need an external ADC to measure analog voltages since the Raspberry Pi does not have its own ADC built-in.
